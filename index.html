TUTORIAL TO USE LOREM PICSUM


To repeat the same picture:

By adding "seed/{seed}" in the middle of the link you can get that same random pic everytime even you refresh.


For example: https://picsum.photos/seed/Alan Harverty/200/300"


To Convert the picture into grayscale:

By adding "?grayscale at the end of the link, you will get a random grayscale image."


For example: "https://picsum.photos/200/300?grayscale"


To make a image look blurry:

By adding "?blur" at the end of the link, you can make any random image blurry.


For example: "https://picsum.photos/200/300?blur"


To adjust the level of blur:

By adding "?blur={number}" at the end of the link, you can adjust the level of blurness


For example: "https://picsum.photos/200/300?blur=2"


To make Hybrid pictures:

We can even extract a picture by combining two above qualities. That is done by using "?grayscale&blur"


For example: "https://picsum.photos/200/300?grayscale&blur=1"


To choose Particular picture.

We can choose a particular pic by adding "id/{img}" in between the link. Below is the example.


For example: "https://picsum.photos/id/22/200/300"
